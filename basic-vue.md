
## Start Vue project with vue cli

$ `vue create project-name`

**manual selection** 

Babel -> Linter -> Unit Testing

Version 3.x -> 
  Eslint + Prettier -> 
    Lint on save -> 
      Jest -> 
        config files -> 
          N for future projects (or Y if you want to have the same settings in future projects)

**how to run the project**

`yarn serve`
